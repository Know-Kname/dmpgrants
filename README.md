# Detroit Memorial Park - Cemetery Management System

A comprehensive business management solution for cemetery operations.

## Quick Start

```bash
npm install
npm run dev
```

## Features

- Work Order Management
- Inventory Tracking
- Financial Management (AR/AP, Deposits, Statements)
- Burial Records
- Contract Management
- Grants & Benefits Tracking
